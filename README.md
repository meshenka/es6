# Build system with webpack and es6 support

## install

```bash
#as root
npm install -g gulp
npm install -g eslint

#as standard user
npm install
```

## Run build process

```bash
#build
npm run build
```

Or
```bash
#build
gulp
```

We can also lint js
```bash
npm run lint
```
